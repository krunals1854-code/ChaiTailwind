# ChaiTailwind 🚀

A lightweight utility-first CSS engine built with JavaScript.

## ✨ Features
- Utility classes like chai-p-20, chai-bg-red
- DOM scanning and dynamic styling
- No CSS required
- Clean DOM (removes classes after applying)

## ⚙️ How it Works
- Scans DOM for chai-* classes
- Parses class names
- Applies inline styles dynamically
